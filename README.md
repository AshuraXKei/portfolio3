# Персональное портфолио разработчика

Одностраничное портфолио (визитная карточка) с секциями **Обо мне / Опыт / Проекты / Контакты**, якорной навигацией и базовой SEO‑оптимизацией.

## Стек

- React + TypeScript
- Vite
- Tailwind CSS

## Быстрый старт

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
npm run preview
```

## Где менять контент

Основные данные лежат в одном месте:

- `src/data/profile.ts`

Там можно поменять:

- имя, роль, описание
- навыки и опыт
- список проектов (карточки автоматически обновятся)
- контакты и ссылки (Telegram/VK/MAX/Yandex Music/Kinopoisk и т.д.)

## Контактная форма

Форма в секции «Контакты» работает без сервера: она формирует `mailto:` ссылку и открывает почтовый клиент пользователя. Это удобно для статического хостинга (например, GitHub Pages).

Есть простая защита от спама (honeypot поле).

## Деплой на GitHub Pages (рекомендация)

Варианты:

1. GitHub Actions + публикация папки `dist/`.
2. Любой статический хостинг (Netlify/Vercel/Cloudflare Pages).

> Для деплоя в подпапку (Project Pages) обычно нужно настроить `base` в конфиге Vite.
